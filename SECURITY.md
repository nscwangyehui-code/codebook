# 安全策略（Security Policy）

感谢你帮助提升《代码伴生笔记（CodeBook）》的安全性。

## 支持范围

我们优先处理影响以下方面的安全问题：
- Electron 主进程 / preload 的权限边界与 IPC 安全
- 路径穿越、任意文件读写、命令执行等本地安全风险
- API Key 泄露、敏感信息意外上传、日志泄露等隐私问题

## 报告漏洞

请不要在公开 Issue 中直接披露可复现的漏洞细节（尤其包含利用方式或 PoC）。

推荐方式：
- 发送邮件到：506816203@qq.com
- 标题：`[SECURITY] 漏洞报告 - 简短描述`
- 内容建议包含：复现步骤、影响范围、环境信息、建议修复方案（可选）

## 响应与披露

我们会在收到报告后尽快确认与评估，并在修复后再进行公开披露与致谢（如你愿意署名）。
