# 更新日志（Changelog）

本项目尽量遵循「Keep a Changelog」的格式：按版本记录新增/变更/修复。  
版本号建议采用语义化版本（SemVer）。

## [0.1.0] - 2026-02-03

### Added
- 三栏工作台：项目树 / 只读代码 / 伴生笔记（可拖拽调宽）
- 同一源文件多笔记 Tab（默认 `main.md`，支持“+ 新建笔记”）
- 笔记自动保存（去抖），笔记强制存放在源代码目录外
- 笔记粘贴图片：保存到 `assets/` 并插入 Markdown 引用
- `Ctrl+P` 关键词检索：找出包含关键词的源代码文件与笔记
- `Ctrl+K` 文件内查找：当前源代码/当前笔记，支持上一处/下一处跳转
- 代码选区右键：插入到当前笔记 / AI 解读选区（可插入笔记）
- 代码版本标记：Git 分支@提交（或非 Git 指纹）
